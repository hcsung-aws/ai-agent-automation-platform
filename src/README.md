# src/ — PoC 구현 (게임 운영 시나리오)

이 디렉토리는 게임 운영 시나리오의 **PoC(Proof of Concept)** 구현입니다.

- `agent/` — DevOps, Analytics, Godot Review, Monitoring Agent
- `tools/` — CloudWatch, EC2, Athena, MMORPG 분석 등 게임 운영 도구
- `utils/` — 실행 로그 유틸리티

루트의 `app.py`, `agentcore_app.py`, `logs_api.py`도 이 PoC용 진입점입니다.

**스타터 킷(범용 템플릿)은 `templates/local/`을 참조하세요.**
