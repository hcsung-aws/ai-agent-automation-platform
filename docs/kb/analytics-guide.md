# Analytics 도메인 가이드

## 개요
게임 운영 데이터 분석을 위한 가이드입니다.

## 주요 분석 영역

### 1. DAU (Daily Active Users)
- 일별 활성 사용자 수 추이 분석
- 이상치 탐지: 전일 대비 20% 이상 변동 시 알림
- 주간/월간 트렌드 비교

### 2. 가챠 확률 분석
- 등급별 확률 검증 (SSR, SR, R, N)
- 기대값 대비 실제 드랍률 비교
- 천장 시스템 도달률 분석

### 3. 재화 흐름 분석
- 획득/소비 밸런스 모니터링
- 인플레이션 지표 추적
- 이상 거래 패턴 탐지

## 분석 도구

| 도구 | 용도 |
|------|------|
| get_dau_trend | DAU 추이 조회 |
| analyze_gacha_rates | 가챠 확률 분석 |
| get_currency_flow | 재화 흐름 조회 |
| detect_anomaly | 이상치 탐지 |

## 알림 기준

| 지표 | 임계값 | 액션 |
|------|--------|------|
| DAU 급감 | -20% | 즉시 보고 |
| 가챠 확률 이상 | ±5% | 검증 요청 |
| 재화 인플레이션 | +30% | 밸런스 검토 |
